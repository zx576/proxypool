API 接口地址 :

http://lab.crossincode.com//proxy/get/

API 接口说明 :

HTTP GET 接口

接口调用频率：3秒1次

举例说明:


获取 5 个 IP : http://lab.crossincode.com//proxy/get/?num=5

获取 5 个 HTTPS 类型的 IP : http://lab.crossincode.com//proxy/get/?num=5&head=https
